# Crit Theme
[![](https://img.shields.io/badge/License-MIT-D3D3D3)](LICENSE.md)

Pathfinder 2e Module for Foundry that allows the GM to set a sound as a character's theme song, that plays automatically whenever they score a critical hit or an enemy critically fails an effect that they created.

## Instructions
  1. In Playlists, there will be a playlist by the name of "Crit Themes". Add all the tracks you wish to use for all critical themes there.
  2. In Game Settings, you can change how long a crit theme should play for, and if other sounds should be disabled while one is playing, as well as if there should be a fade-in.
  3. Open the character sheet of whichever character you want to assign a crit theme. There will be a button labeled "🎵Crit Theme" in the top bar of the window, near the configure and prototype token buttons.
  4. Choose the desired track from the dropdown.
